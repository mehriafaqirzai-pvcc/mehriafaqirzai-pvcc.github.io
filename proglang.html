Programming Languages

Terms Definitions
Machine Language Low-level programming language directly understandable by a computer's CPU.
Debugging The process of finding and fixing errors (bugs) in a computer program.
Assembler A program that translates assembly language code into machine code.
Assembly Language A low-level programming language that is specific to a particular computer architecture.
Translator A generic term for a program that translates code from one programming language into another.
Compiler A program that translates high-level source code into machine code all at once.
Interpreter A program that translates high-level source code into machine code and executes it line by line.
Cross-platform Software Software that can run on different types of computer platforms without modification.
Natural Language A human language used for communication, such as English or Spanish.
Formal Language A language with a precise set of rules for writing and interpreting code, often used in programming.
Machine Independence The ability of a program to run on different types of computer hardware without modification.
High-level Programming Language A programming language that is easier for humans to understand and write compared to low-level languages.
Object-oriented Programming (OOP) A programming paradigm based on the concept of "objects" that can contain data and code.
Scripting Language A programming language used for writing scripts, often interpreted rather than compiled.
Data Types Categories that define the type of data that a variable can hold, such as integer, float, and string.
Primitive Data Type A basic data type that is not composed of other data types.
Data Structure An organized way of storing and managing data, such as arrays, lists, and dictionaries.
Python Collection Data Structures Specific data structures provided by the Python programming language, including list, tuple, set, and dictionary.
Function A named section of a program that performs a specific task.
Local Variable A variable that is declared and used only within a specific function or block of code.
Global Variable A variable that is accessible and can be modified throughout the entire program.
Source Program The original human-readable code written by a programmer.
Object Program The output of the compiler or assembler, typically in machine code.
