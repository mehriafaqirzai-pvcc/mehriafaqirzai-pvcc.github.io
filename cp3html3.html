Terms and Definitions
Operating System
Software that manages computer hardware and provides services for computer programs.
Job Queue
A job queue is a list of tasks or jobs awaiting execution by a computer system, often used in batch processing environments.
ETFO:
ETFO: ETFO is not a standard term in the context of operating systems or computer science. It might be a specific acronym or abbreviation related to a particular system or organization.
Interactive Processing
Interactive processing refers to a mode of computer operation where the user can interact with the system in real-time, providing input and receiving immediate responses.
Time-sharing:
Time-sharing is a technique that allows multiple users or processes to share a single computer's resources, enabling them to run concurrently.
Multiprogramming:
Multiprogramming is a technique in which multiple programs are loaded into memory at the same time, allowing the CPU to switch between them for execution.
Application Software:
Application software, also known as apps, refers to computer programs designed to perform specific tasks or functions for end-users.
System Software:
System software is software that manages and controls the computer hardware and provides a platform for application software to run.
Utility Software:
Utility software consists of programs that assist in managing, maintaining, and optimizing the computer system, such as antivirus software or disk cleanup tools.
Graphical User Interface (GUI):
A GUI is a visual way for users to interact with a computer or software using graphical elements like windows, icons, buttons, and menus, rather than text-based commands.
Shell:
In the context of operating systems, a shell is a command-line interface or a program that provides a user with a way to interact with the operating system.
Kernel:
The kernel is the core component of an operating system that manages hardware resources and provides essential services for other software.
Device Driver:
A device driver is software that enables the operating system to communicate with and control hardware devices like printers, graphics cards, or storage drives.
File Manager:
A file manager is software that allows users to organize, navigate, and manipulate files and directories on a computer.
Memory Manager:
The memory manager in an operating system is responsible for managing the computer's memory, allocating and deallocating memory as needed.
Virtual Memory:
Virtual memory is a memory management technique that uses disk space to simulate additional RAM, allowing programs to run as if they have more memory than physically available.
Bootstrapping:
Bootstrapping refers to the process of loading and initializing the operating system on a computer when it is powered on.
ROM (Read-Only Memory):
ROM is a type of computer memory that stores data that cannot be modified. It typically contains firmware or essential software instructions.
Deadlock:
Deadlock is a situation in which two or more processes are unable to proceed because each is waiting for the other to release a resource.
Spooling:
Spooling (Simultaneous Peripheral Operations On-Line) is a technique used to queue and manage data or print jobs so that the computer can process them in an efficient order.
Super User (or Administrator):
A super user, often called an administrator, is a user with elevated privileges in an operating system, allowing them to perform administrative tasks and configurations.
Multi-Factor Authentication:
Multi-factor authentication (MFA) is a security process that requires users to provide multiple forms of identification, typically a combination of something they know (e.g., a password) and something they have (e.g., a fingerprint or security token) to access a system or service, enhancing security.
