Chapter 4: Networking & The Internet

Term Definition
Network classifications
  • LAN (Local Area Network): A network that is limited to a small geographic area, such as a single building or a campus.
  • MAN (Metropolitan Area Network): A network that covers a larger geographic area than a LAN, such as a city or a large campus.
  • WAN (Wide Area Network): A network that covers a broad area, often a country or continent, using various communication channels.
Network models
  • Bus network: A network topology where all devices share a common communication medium.
  • Star network: A network topology where all devices are connected to a central hub or switch.
Protocols
  • CSMA/CD (Ethernet): A protocol used in Ethernet networks to manage access to the communication medium by devices.
  • CSMA/CA (Wifi): A protocol used in wireless networks to manage access to the communication medium by devices.
Combining networks
  • Repeater: A device that amplifies and retransmits signals to extend the reach of a network.
  • Bridge: A device that connects and filters traffic between two or more network segments at the data link layer.
  • Switch: A device that connects devices within the same network and uses MAC addresses to forward data to the correct destination.
  • Router: A device that connects different networks and forwards data between them based on IP addresses.
  • Gateway: A device that connects different types of networks and facilitates communication between them.
Client/server model a distributed application framework dividing tasks between servers and clients, which either reside in the same system or communicate through a computer network or the Internet.
P2P: peer-to-peer a decentralized communications model in which each party has the same capabilities and either party can initiate a communication session.

Cybersecurity
  • Malware: Software designed to harm or exploit computer systems, including viruses, worms, and spyware.
  • Virus: A type of malware that attaches itself to a host program and spreads when the host program is executed.
  • Worm: A standalone malware that replicates itself and spreads to other computers without needing a host program.
  • Trojan horse: A type of malware that disguises itself as legitimate software to trick users into installing it.
  • Spyware: Software that secretly collects user information without their knowledge or consent.
  • Phishing: A method of fraudulently obtaining sensitive information by pretending to be a trustworthy entity in electronic communication.
  • DDoS attack: Distributed Denial of Service attack, where multiple systems overwhelm the bandwidth or resources of a targeted system.
Protection
  • Firewall: A security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
  • Spam filter: A program or service that detects and filters out unwanted or unsolicited email (spam).
  • Proxy server: An intermediate server that acts as a gateway between a user's device and the internet, enhancing security and performance.
  • Antivirus software: Software designed to detect, prevent, and remove malicious software from a computer or network.
  • Cryptography: The practice and study of techniques for secure communication in the presence of third parties.
Public key encryption is a method of encrypting or signing data with two different keys and making one of the keys, the public key, available for anyone to use.
